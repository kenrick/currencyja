Currency Jamaica

Currency Jamaica, a simple way to keep track of how much the JMD is being traded for and where to get the best price.

Download .zip Download .tar.gz View on GitHub

Currency Jamaica API.

This will provide a simple interface over HTTP for accessing the data.

Supported Traders / Cambio

  • Scotiabank
  • National Commercial Bank
  • Jamaica National Building Society
  • FX Trader

Supported Currencies

  • USD
  • CAD
  • EUR
  • GBP

Endpoints

This route will return all the traders and all the currencies.

http://currencyja.com/api/traders.json

This route will return all the traders with the given currency.

http://currencyja.com/api/traders/(:currency).json

Options: usd cad eur gbp
